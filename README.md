# Fortune Wheel Randomizer

A simple browser-based fortune wheel application that lets users enter a list of values, spin an animated wheel, and randomly select one result.  
Built as a single static HTML file, it is easy to deploy on GitHub Pages and use in any modern browser.

## Features

- Enter multiple values separated by commas or line breaks.
- Generate a colorful wheel with equal slices.
- Spin the wheel with smooth animation.
- Show the selected result clearly on screen.
- Reset and rebuild the wheel anytime.
- Responsive design for desktop and mobile browsers.

## How It Works

1. Enter your values in the input area.
2. Click **Create Wheel** to build the wheel.
3. Click **Spin** to animate the wheel.
4. The wheel stops on one random value and displays the result.

## Local Usage

1. Download or clone the repository.
2. Open `index.html` in a browser.
3. Enter values and start spinning.

## Deploy on GitHub Pages

1. Create a GitHub repository.
2. Add `index.html` to the repository root.
3. Commit and push the changes.
4. Open the repository settings.
5. Go to **Pages**.
6. Select **Deploy from a branch**.
7. Choose the `main` branch and `/ (root)` folder.
8. Save and wait for GitHub Pages to publish the site.
